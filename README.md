# 大家在一起 together

这是一个根据上海交大学期时间排的课程时间表，只关注时间本身，不关注具体课程，旨在发现一群同学的空闲时间，方便安排生活、工作。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploy
```
npm run deploy
```
